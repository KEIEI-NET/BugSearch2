---

🎯 インストール先:

Windows: %APPDATA%\claude-code\aliases.md
Mac:     ~/Library/Application Support/claude-code/aliases.md
Linux:   ~/.config/claude-code/aliases.md

---

✨ 提供されるエイリアス:

@perfect  - 完全品質保証（静的100点 + テスト100%）
@tdd      - テスト駆動開発ワークフロー
@validate - 既存コード検証
@quick    - 高速プロトタイピング

---

🚀 クイックスタート:

# Windows
cd Install_PC
install_windows.bat

# Mac/Linux
cd Install_PC
chmod +x install_unix.sh
./install_unix.sh

# 確認
verify_install.bat  (Windows)
./verify_install.sh (Mac/Linux)

---

📚 使用例:

@perfect ユーザー認証機能を実装して
→ コード実装 → 静的解析100点 → テスト100% → コミット

@tdd 決済処理ロジックを実装して
→ テスト作成 → 実装 → リファクタリング → 100点達成

@validate apply_improvements_from_report.py
→ 品質スコア → テスト実行 → レポート生成

@quick CSVから売上グラフ作成
→ 素早く実装 → 品質チェックスキップ
